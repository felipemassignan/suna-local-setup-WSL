# Guia de Instalação Detalhado

## Pré-requisitos

### Sistema Windows
- Windows 10 versão 2004+ ou Windows 11
- WSL2 habilitado e configurado
- 8GB+ RAM (recomendado 16GB)
- 50GB+ espaço livre em disco
- PowerShell com privilégios administrativos

### Verificar WSL2
```powershell
wsl --version
wsl --list --verbose
